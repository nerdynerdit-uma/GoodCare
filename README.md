# GoodCare Healthcare - Luxury Home Care Website

A luxurious, modern website for GoodCare Healthcare, a premium caregiving agency providing compassionate home care services for seniors.

## Features

- **Modern, Luxurious Design**: Elegant and professional design inspired by luxury healthcare websites
- **Fully Responsive**: Works seamlessly on desktop, tablet, and mobile devices
- **Smooth Animations**: Subtle animations and transitions for enhanced user experience
- **Interactive Navigation**: Fixed navbar with smooth scrolling and active section highlighting
- **Service Showcase**: Comprehensive display of home care services
- **Contact Form**: Functional contact form for inquiries and consultations
- **Testimonials**: Client reviews section to build trust
- **SEO Friendly**: Semantic HTML structure for better search engine optimization

## Services Highlighted

1. **Live-In Home Care** - Round-the-clock care with a dedicated caregiver
2. **24-Hour Home Care** - Continuous care with rotating caregivers
3. **Hourly Home Care** - Flexible care schedules
4. **Dementia & Memory Care** - Specialized care for Alzheimer's and dementia

## Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with CSS Grid, Flexbox, and custom properties
- **JavaScript (Vanilla)**: Interactive features and animations
- **Font Awesome**: Icons for visual enhancement
- **Google Fonts**: Playfair Display (headings) and Inter (body text)

## Getting Started

1. Clone or download this repository
2. Open `index.html` in a web browser
3. No build process or dependencies required - works out of the box!

## Customization

### Colors
The color scheme can be customized in `styles.css` using CSS custom properties:

```css
:root {
    --primary-color: #1a4d3a;      /* Main brand color */
    --primary-dark: #0f3324;      /* Darker shade */
    --accent-color: #d4af37;       /* Accent/gold color */
    --text-dark: #1a1a1a;          /* Dark text */
    --text-light: #666666;         /* Light text */
}
```

### Content
- Update contact information in the contact section
- Modify service descriptions in the HTML
- Add/remove review cards as needed
- Update footer links and information

### Images
To add background images or photos:
1. Create an `assets/images/` folder
2. Add your images
3. Update the CSS `background-image` properties or add `<img>` tags in HTML
